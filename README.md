
# Portfólio Diego Santos de Araújo

Portfólio pessoal moderno, responsivo e multilíngue (PT/EN), desenvolvido com React, Vite, TypeScript, TailwindCSS, shadcn/ui e Framer Motion. Apresenta trajetória, projetos, stack, formação e contato de forma elegante e animada.

## 🚀 Tecnologias principais
- React 18
- Vite
- TypeScript
- TailwindCSS
- shadcn/ui (Radix)
- Framer Motion

## 📁 Estrutura principal
```
src/
	App.tsx            # Contexto de idioma, providers e rotas
	main.tsx           # Ponto de entrada
	index.css          # Estilos globais e tokens
	components/        # Seções: Hero, About, Stack, Projects, Experience, Education, Contact, Footer, Navbar
	pages/             # Páginas (Index, NotFound)
	hooks/, lib/       # Hooks e utilidades
```

## 🖥️ Como rodar localmente
1. Instale as dependências:
	 ```
	 npm install
	 ```
2. Inicie o servidor de desenvolvimento:
	 ```
	 npm run dev
	 ```
3. Acesse http://localhost:8080 no navegador.

## 🛠️ Scripts úteis
- `npm run dev` — inicia o servidor local
- `npm run build` — gera build de produção
- `npm run lint` — verifica padrões de código
- `npm test` — executa testes automatizados

## 📦 Requisitos
- Node.js 18+
- npm 9+ (ou bun/yarn compatível)

## ✨ Diferenciais e funcionalidades
- **Navbar fixa** com troca de idioma (PT/EN) integrada
- **Página única** com seções: Hero, Sobre, Stack, Projetos, Experiência, Formação, Contato e Footer
- **Animações suaves** e responsivas com Framer Motion
- **Cards e timeline** para projetos, experiência e formação
- **Contato com links sociais** e botão de agendamento
- **Design moderno** com glassmorphism, gradientes e responsividade total
- **Tradução dinâmica** em todas as seções principais

## ⚠️ Pontos de atenção
- Alguns links (currículo, agendamento, projetos demo) são placeholders — personalize para seu uso
- Dados de contato devem ser atualizados para informações reais
- O projeto inclui providers e dependências extras (ex: React Query) que podem ser removidos se não forem usados
